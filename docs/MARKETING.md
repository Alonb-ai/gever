# גבר — תכנית שיווק והפצה (GTM)

נספח שיווקי לאפיון. מתעד את ערוצי ההפצה והנכסים לבנייה. מסמך עבודה —
ניתן לייצא ל‑`גבר_MVP_Spec.docx` לגרסה למשקיעים.

## מיצוב

- **המסר:** "יש לך גבר שיעשה את זה." שולחים הודעה אחת — גבר סוגר.
- **קהל:** 25‑45, ישראל, ‏WhatsApp‑native‏, שונאים בירוקרטיה דיגיטלית.
- **למה עכשיו:** הבוט עושה את העבודה במקום המשתמש, בלי אפליקציה ובלי טפסים.

## ערוצים

### אורגני
- **Instagram** (‏handle‏ TBD): ‏reels‏ קצרים — "גבר סוגר X", הומור ישראלי, דמו אמיתי.
- **Twitter / X**: ‏build in public‏ — ‏thread‑ים‏, התקדמות, דמו. בונה אמון ומשיכה מוקדמת.

### ממומן (paid)
- **Meta Ads** (‏Instagram + Facebook‏): קהל ישראלי, ‏retargeting‏ ללנדינג/וייטליסט.
  תקציב טסט קטן → אופטימיזציה ל‑‏CTR‏ ואז ל‑‏signups‏.
- **TikTok** (אופציונלי): אם פורמט הסרטון האנכי עובד.

### הפצה אישית (IRL)
- רשת החברים של המייסד — הבטא הראשון, פידבק איכותי, ‏word of mouth‏. הכי חזק בשלב מוקדם.

## נכסים לבנייה

### סרטון דמו (Remotion)
- אנימציית שיחת ‏WhatsApp‏ עם גבר שסוגר משימה ‏(הזמנת שולחן)‏ ‏end‑to‑end‏.
- פורמט: אנכי ‏9:16‏, ~20‑30 שניות — מתאים ל‑‏Reels/TikTok‏ ולהטמעה בלנדינג.
- ‏Remotion‏ ‏(React)‏ — קל לייצר וריאציות לכל ערוץ ולכל סוג משימה.
- מראה את ה‑‏UX‏ האמיתי: הודעה → "על זה אחי" 🔄 → "סגור ✅".

### דף נחיתה (לבנות)
- מטרה ראשונית: וייטליסט / ‏signup‏ + הטמעת סרטון הדמו.
- ‏CTA‏ ראשי: ‏click‑to‑WhatsApp‏ (‏wa.me‏ link) — "שלח לגבר עכשיו".
- תוכן: המסר, 3 דוגמאות שימוש, הסרטון, מחיר ‏(בהמשך)‏, ‏social proof‏.
- טכנולוגיה: פשוט — ‏Next.js / סטטי‏, אירוח ב‑‏Vercel‏.

### לוגו + מותג (לעצב)
- וייב: ישראלי, גברי, סחבקי, ביטחון רגוע — תואם לאישיות הבוט.
- שם המותג: **גבר**. צבעים/טיפוגרפיה: TBD.
