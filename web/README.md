# web — דף נחיתה של גבר

`index.html` הוא דף נחיתה סטטי, עצמאי, ללא build (תלות חיצונית יחידה: Google Fonts).
תוכן הדף מיושר לבטא הסגורה: מסעדות בלבד, עצירה לפני שלב האשראי, בלי תמחור ובלי
social proof מומצא. הטוקנים העיצוביים מתועדים ב-`docs/marketing/design-tokens.md`.

**Deploy (GitHub Pages):** אוטומטי. push ל-`main` שנוגע ב-`web/**` מפעיל את
`.github/workflows/pages.yml` שמפרסם את תיקיית `web/` ל-
`https://alonb-ai.github.io/gever/`. אפשר גם להריץ ידנית (workflow_dispatch)
מטאב Actions. אין שלב build.
